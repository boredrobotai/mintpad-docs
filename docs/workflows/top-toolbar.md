# Top Toolbar

## Overview

The top toolbar provides quick access to global MintPad controls, wallet session status, balances, settings and shutdown actions.

These controls are available from the main MintPad workspace.

## Toolbar Overview

![Top Toolbar](../assets/images/settings/top-toolbar-01-overview.png)

The toolbar includes:

- Gas reference display
- ETH and BASE price references
- Balance refresh
- Vault lock status
- Vault unlock
- Settings
- Logout
- Turn Off

## Vault Controls

![Vault Controls](../assets/images/settings/top-toolbar-02-vault-controls.png)

The Vault indicator shows whether the Wallet Vault is currently locked or unlocked.

Use **Unlock** before any operation that requires transaction signing.

Use **Lock Vault** when you want to keep MintPad open but lock wallet signing access.

## Settings, Logout and Turn Off

The toolbar also includes the main session controls:

- **Settings** — opens the Runtime Settings modal.
- **Logout** — ends the current MintPad session.
- **Turn Off** — safely closes MintPad and the local backend.

## Balance Refresh

Use **Balances** to refresh wallet balance information when needed.

This is useful before minting, dispersing funds or checking whether wallets have enough funds for gas.

## Notes

**Note:** Transaction signing requires the Wallet Vault to be unlocked.

**Note:** Turn Off is the preferred shutdown action when you are finished using MintPad.

**Note:** If the UI balance display looks stale, refresh balances before creating or sending tasks.
